A collection of my work spanning machine learning and data science. Each project represents a learning journey and demonstrates different aspects of my technical skills.

Building Heat Load Analysis

Python reimplementation of a research-grade pipeline for building heat-load characterization, migrating an R methodology to an HPC-ready, reproducible Snakemake workflow. It models hourly energy behavior with changepoint signatures, clusters typical daily patterns with K-means, and trains CART models to produce prediction-ready outputs and rich diagnostics.

Python Pandas Scikit-Learn JobLib DEAP Optuna
GitHub

Predictions Changepoint

A production-ready Python system for forecasting building energy loads using a two-stage ML pipeline: it first classifies days into clusters from weather and temporal features, then predicts hourly loads with cluster-specific linear models. It supports batch predictions with scikit-learn .pkl models and CSV parameters, and is grounded in peer‑reviewed changepoint, clustering, and CART research.

Python Scikit-Learn Pandas
GitHub

Simpsons DaVinci

Simpsons-DaVinci is a comprehensive deep learning project that implements and evaluates multiple Generative Adversarial Network (GAN) architectures, including DCGAN, Wasserstein GAN, Autoencoder-GAN, and StyleGAN2, to generate synthetic images in The Simpsons' distinctive cartoon style, providing insights into training stability, image quality, and architectural performance across different generative models.

Python PyTorch GAN
GitHub

Movie Recommender
